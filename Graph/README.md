# Graph Algorithm

Implementation of a Directed Graph algorithm in C++ language. <br>
Supports various operations on the edges and nodes of the graph.

### Operations :
- createNode
- cloneNode 
- removeNode
- createEdge
- getNodeAtLabel
- addTargetsOfOther
- addSourcesOfOther
- unionize
- merge
- takeLabels
- createDotFile
- printGraph
